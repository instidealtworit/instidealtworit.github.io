1. ADDING A INSTITUTION

STEP 1 : open the html document

iit.html
medical.html
cbse.html
chse.html
in notepad depending upon the category of institute.

STEP 2 : scroll down and you will find a "div" element with class list.Inside this div element ,you will find div with class="classes".



STEP 3 : copy this div with class="classes" and paste just below the last div with class="classes".

STEP 4 : change the image of new institute in "img" tag . NOTE: The image must be placed in images folder.

STEP 5 : change the courses offered if any and change the link in "href" tag to page of new institution that we are going to create in next steps.

CREATING A NEW PAGE FOR INSTITUTION


STEP 6 : create a copy of any institution page for ex- vertex.html.Change the name of file with new institution.Open this file in notepad

STEP 7 : scroll down and you will find a comment "content starts" just after the nav ends.



STEP 8 : Change the heading from vertex classes to new institution and image in img tag just below heading.



STEP 9 : Again change the heading from vertex classes to new institution and in the div with class="about" , replace the content of new institution.

STEP 10 : just below it, you will find a div with class="info" , replace vertex classes with new institution.

STEP 11 : just below it, you will find a div with class="about" , replace the lines of vertex classes with new institution.

STEP 12 : To change the title of the page,go the head section and change title in title tag.

STEP 13 : save this file and finally its done.




2. ADD A NEW ITEM IN NAVIGATION MENU



STEP 1 : open the html file and look for the "nav" tag.(in the beginning of body)



STEP 2 : To add the item in center ,copy the li element just below the contact us link and replace the link in "href" tag to new page.

STEP 3 : To add the item in right corner ,copy the li element in ul element with class navbar-right and replace the link in "href" tag to new page.




3. CHANGE THE BACKGROUND PIC



STEP 1 : open the css file with name "instideal.css".



STEP 2 : In the .frame styling, change the url in backgrounnd-image property to the new pic.

STEP 3 : Background pic is changed.
